系所簡介


本系教學目標、理念及特色

圖書資訊學系發展方向與輔仁大學校務發展的方向是一致的。輔仁大學創校之宗旨在於追求真、善、美、聖全人教育之師生共同體，致力於中華文化與基督信仰之交融，獻身於學術研究與弘揚真理，以促使社會均衡發展及增進人類福祉。學校在學術研究方面，追求真知力行，培育人文精神，推動知識整合；而培育人才方面，兼具通才與專才養成教育，尤其致力人文精神的培養，期使學生在就業之餘，能有高尚的情操與豐富的人生。

本系的教學目標在於：(1) 提升學生圖書資訊學理論與實務之知識與能力、 (2) 提升學生相關電腦科技理論與實務之知識與能力與(3) 提升學生於數位內容規畫、管理與傳播之知識與能力。在這些教學目標完成後，學生未來能夠具備圖書資訊服務的能力，融合科技的運用，培育人文精神，推動知識整合，並鼓勵學生體驗服務社會，貢獻所學知識。這與學校校務發展的方向與理念是一致的。


大學部：

理念：

本系配合學校教育宗旨與目標，訓練學生蒐集、處理、評估及傳播知識與資訊的能力，以培養圖書資訊及相關資訊科技之專業人才。本系教學目標的落實，在於讓學生未來能夠具備圖書資訊服務的能力，並更具有圖書資訊領域職場之競爭能力。輔仁大學治學之目標就學術層面而言，在於專精學術研究，追求真知力行；培育人文精神，推動知識整合。本系之治學理念也遵循這樣的方向。另外就團體意識層面，本系師生融洽，經常透過正式與非正式之活動發展良性互動，增進羣己關係之平衡。這些都與學校的目標互相呼應。

特色：

學生歷年來積極的參與圖書館服務的工作，也成為本系發展的特色之一。透過學生自發性的服務活動，一方面學以致用，另一方面讓學生體驗服務社會，貢獻所學知識。學生能夠藉由這些自導性的社會服務，體會圖書資訊知識的價值。這種以實際行動，運用專業所學服務人群的特色與輔仁大學校務發展的方向是一致的。雖然並非以任何宗教的名義，但其宗旨皆在於促使社會均衡發展及增進人類福祉。


碩士班：

理念：

本系成立碩士班之目的，乃希望圖書資訊學相關科系畢業人才能接受更深廣的教育，培養專業研究素養與能力，並期能吸收各種不同學科背景的人才，為圖書資訊界培育具多元化知識與技能的人才。碩士班之專業訓練，尊重學生本身的學術專長與興趣，教師積極熱心的指導學生完成碩士論文。這些都與學校尊重學術自由目標、重視各科專業倫理互相呼應。

特色：

探討「圖書館學」與「資訊科學」之間，明確而具體的關係，以及二者相輔相成之道。整合所有科目內容，並利用現代科技充實傳統學科之教學內容。學術研究方面，追求真知力行，融合科技的運用，培育人文精神，推動知識整合。另外由部份碩士班學生的論文主題可見本系碩士生在學術研究與學校宗旨目標（人性尊嚴、人生意義、學術研究、團體意識、文化交流、宗教合作與服務人群）呼應之處。結合校務發展與教育目標之相關碩士研究論文主題，都可看出本系專業與社區服務結合之情形。
